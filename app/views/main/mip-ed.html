MIP[ed]

MIP[ed] -- integrated -- adalah platform yang terintegrasi untuk membuat aplikasi, berdasarkan JavaEE (Kami menyebutnya JavaEE 7+, karena kita menambahkan beberapa konfigurasi untuk membuatnya dapat berjalan pada kontainer servlet). MIP[ed] memiliki 2 tipe database penyimpanan default, SQL berdasarkan JPA dan BigData berdasarkan Hadoop/HBase.

MVC

Metode untuk membuat sebuah aplikasi dengan memisahkan data dari tampilan dan cara bagaimana memprosesnya.

Role

Metode untuk menangani pembagian hak akses pengguna module dalam suatu web.

Security Integration

Sistem yang terintegrasi untuk menangani masalah keamanan software.

RESTfull

Web service yang di implemantasikan menggunakan http dengan prinsip-prinsip REST.

JAX-RS

Bahasa pemrograman Java API yang menyediakan dukungan dalam membuat layanan web sesuai dengan REST.

OAuth Client

Standar otorisasi aplikasi client untuk mengakses web server.
